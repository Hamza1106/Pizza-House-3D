🚀 Getting Started
## Installation

Clone the repository:

```bash
git clone https://github.com/Hamza1106/pizza-house-3d.git
```

Go to the project directory:

```bash
cd pizza-house-3d
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
